# Set 3 — Broadcasting


Broadcasting lets NumPy “stretch” arrays of different shapes to make elementwise operations work without copying.

## Tasks
1) Row-normalize using `keepdims=True`  
2) Pairwise squared distances using `A[:, None, :] - B[None, :, :]`
